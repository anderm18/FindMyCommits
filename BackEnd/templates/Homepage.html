FindMyCommits API

You've reached the FindMyCommits API Homepage. Below are the different endpoints available:

GET / - The homepage of the API (You're currently here)

POST /getcommits - Takes 3 Body Form-Data Keys: name, link, and token. Returns a JSON listing every commit the given name in the given link (repository).

POST /getsrc - When reached, it will pull code from the FindMyCommits repository on GitHub (used for CI)
